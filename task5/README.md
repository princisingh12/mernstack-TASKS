# Task-5: User Authentication System using LocalStorage

## Objective
Build a simple authentication system using HTML, CSS, and JavaScript.

## Features
- User Registration
- Login System
- LocalStorage for data storage
- Form validation
- Protected Home Page
- Logout functionality
- Show/Hide password feature

## Tech Stack
- HTML
- CSS
- JavaScript

## How it works
1. User registers → data saved in localStorage
2. User logs in → credentials verified
3. On success → redirected to Home page
4. Logout clears session

## Bonus Features
- Prevent direct access to Home page
- Show/Hide password toggle
